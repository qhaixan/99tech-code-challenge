## Technology Stack

- **Frontend**: React, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, PostCSS
- **Package Management**: npm

## Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/qhaixan/99tech-code-challenge.git
   cd 99tech-code-challenge
   ```

2. **Install dependencies** (for each problem directory):
   ```bash
   cd problem-2
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
